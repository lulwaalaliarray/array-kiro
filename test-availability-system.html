Testing Instructions

For Doctors:

  1. Log in as a doctor account
  2. Go to "Manage Availability" from the dashboard
  3. Set your weekly schedule (enable/disable days, set hours)
  4. Click "Save Availability"
  5. Verify the settings are saved by refreshing the page

For Patients:

  1. Go to "Find Doctors" and select a doctor
  2. Click "View Profile" then "Book Appointment"
  3. Try selecting different dates - unavailable dates will show error
  4. For available dates, only see time slots within doctor's hours
  5. Complete booking and verify appointment is created
